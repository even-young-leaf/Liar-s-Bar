## 摘要

本项目设计并实现了一个名为 Liar's Bar Online 的多人在线实时博弈平台。系统采用前后端分离架构：Vue 3 负责页面展示与用户交互，Go/Gin 提供 REST API 和 WebSocket 网关，MySQL 保存用户与房间等持久化数据，Redis 负责缓存与队列能力，Python/FastAPI 提供 AI 推理和训练原型。游戏服务端维护权威状态，支持注册登录、房间管理、快速匹配、AI 超时补位、实时聊天、出牌与质疑、轮盘惩罚以及胜负结算。项目通过 Docker Compose 编排六个服务，并使用 Nginx 统一代理 HTTP 与 WebSocket 流量。实际测试表明，系统可完成从注册登录、进入大厅、快速匹配到 AI 补位开局的完整网络交互流程。

**关键词**: 网络编程；WebSocket；Go；Vue 3；Docker Compose；实时游戏；PPO
